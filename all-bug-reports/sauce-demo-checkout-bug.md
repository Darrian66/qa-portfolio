Название: Поле "Zip/Postal Code" принимает буквы и спецсимволы на странице Checkout

Окружение: Windows 11
Браузер: Mozilla Firefox 146.0.0.383
Сайт: https://www.saucedemo.com/
Учётная запись: стандартный пользователь (standard_user / secret_sauce)

Предусловия:
Пользователь авторизован в системе.

Шаги воспроизведения:
1. Перейти на сайт https://www.saucedemo.com/
2. Авторизоваться валидными данными (standard_user / secret_sauce)
3. Добавить любой товар в корзину
4. Перейти в корзину
5. Нажать кнопку Checkout
6. Ввести валидные данные в поля First Name и Last Name
7. В поле “Zip/Postal Code” ввести буквы, спецсимволы или их комбинацию(например: qwE!!34)
8. Нажать кнопку Continue

Ожидаемый результат:
Появляется сообщение об ошибке ввода (Validation Error).
Переход на следующий шаг оформления заказа блокируется.

Фактический результат:
Система принимает некорректные данные, пользователь переходит на страницу подтверждения заказа (checkout-step-two).

Приоритет: Very High
