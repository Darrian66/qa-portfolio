 Баг: Неверная ссылка/домен на оффициальном сайте Tiktok.com

 Описание: При нажатии кнопки на сайте тиктока вместо переходящего сайта выходит проблема с подключением к сайту

 Окружение: Windows 11 / Firefox 145.0, Firefox 145.0(Incognito), Edge 142.0.3595.80

 Шаги для воспроизведения:
1. Открыть TikTok в браузере
2. Убедиться что вы в UA или RU регионе
3. Слева в меню кнопок нажать кнопку "Еще" с рисунком троеточья
4. Нажать на кнопку "Продажа в TikTok Shop"

Ожидаемый результат:
Переход на сайт seller-ua.tiktok.com

Фактический результат:
Ошибка при его включении "DNS_PROBE_FINISHED_NXDOMAIN"

Severity: Medium
