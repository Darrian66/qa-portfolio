# Test Cases — TikTok Web: Раздел «Лайки»

TC-01: Проверка отображения фото
Precondition: Пользователь авторизован, есть лайкнутые фото  
Steps:
1. Перейти в раздел «Лайки»
2. Открыть любое фото
Expected result: Фото отображается полностью, без искажений

---

TC-02: Проверка статичности фото при выключенной автопрокрутке
Precondition: Автопрокрутка выключена  
Steps:
1. Открыть лайкнутое фото
2. Не взаимодействовать с экраном
Expected result: Фото остаётся статичным, не листается

---

TC-03: Проверка ручного листания фото
Steps:
1. Открыть фото в разделе лайков
2. Пролистать мышью / свайпом
Expected result: Фото листается вручную без задержек

---

TC-04: Проверка запуска видео (автостарт выключен)
Precondition: Автостарт выключен
Steps:
1. Открыть лайкнутое видео
Expected result: Видео не начинается автоматически

---

TC-05: Проверка автостарта при включённой автопрокрутке
Precondition: Автопрокрутка включена  
Steps:
1. Открыть лайкнутое видео
Expected result: Видео запускается автоматически

---

TC-06: Проверка автопрокрутки фото при выключенной автопрокрутке
Precondition: Автопрокрутка выключена  
Steps:
1. Открыть фото
2. Ждать 10 секунд
Expected result: Фото НЕ должно пролистаться

---

TC-07: Проверка автопрокрутки на разных браузерах
Steps:
1. Открыть TikTok Web в Chrome / Firefox / Edge / Safari
2. Перейти в раздел лайков
3. Открыть фото или видео
Expected result: Поведение одинаковое во всех браузерах

---

TC-08: Проверка отображения кнопок взаимодействия
Steps:
1. Открыть любое лайкнутое фото или видео
2. Проверить наличие кнопок Like, Share, Comment
Expected Result: Все кнопки отображаются корректно и доступны для взаимодействия

---

TC-09: Проверка взаимодействия с кнопкой Like:
1. Открыть видео в разделе Likes
2. Убрать лайк на видео (кнопка должна стать серой)
3. Вернуть лайк (кнопка должна стать красной)
Expected result: Кнопка лайка работает корректно

---

ТС-10: Проверка работы кнопки Share:
1. Открыть видео
2. Нажать на иконку Share
Expected result: появляется всплывающее меню с разными способами поделиться видео

---

ТС-11: Проверка в режиме инкогнито:
Precondition: Убедиться что режим инкогнито включён, автопрокрута выключена
1. Открыть фото в разделе Likes
2. Ждать 5-7 секунд
Expected result: Фото НЕ должны пролистыватся автоматически
