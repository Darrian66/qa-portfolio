Баг: Автоматическая прокрутка фото и автозапуск видео при выключенной автопрокрутке

Описание: В Tiktok Web фото автоматически пролистываются, а видео запускаются автоматически, несмотря на отключённую автопрокрутку.

Окружение:
Windows 11 / Firefox 145.0(Incognito), Edge 142.0.3595.80

Iphone / Safari

Шаги для воспроизведения:
1. Открыть TikTok в браузере
2. Перейти в профиль > лайки
3. Открыть любое фото в полноэкранном виде
4. Убедиться что автопрокрутка отключена
5. Наблюдать поведение

Ожидаемый результат:
Фото и видео должны оставаться статичными до ручного действия пользователем.

Фактический результат:
Фото пролистываются автоматически, следующее видео включается автоматически

Severity: Meduim
Priority: Medium
