Чек-лист — TikTok Web: Раздел «Лайки»

Отображение фото и видео
* Фото отображается полностью и без искажений
* Видео отображается корректно
* Открытое фото остаётся статичным при выключенной автопрокрутке
* Фото листается вручную без задержек
* Видео автоматически запускается при включённой автопрокрутке

Кроссбраузерность
* Поведение фото и видео одинаковое в Chrome
* Поведение одинаковое в Edge
* Поведение одинаковое в Firefox
* Поведение одинаковое в Safari
* Автопрокрутка работает одинаково во всех браузерах

Отображение UI
* Кнопка Like отображается
* Кнопка Share отображается
* Кнопка Comment отображается
* Все кнопки доступны для нажатия

Функциональность кнопок
* Кнопка Like корректно меняет состояние (красная и серая)
* Лайк успешно ставится
* Лайк усппешно снимается
* Кнопка Comment работает корректно
* Кнопка Share вызывает всплывающее меню с возможными вариантами поделиться

Режим Инкогнито
* Фото НЕ листаются автоматически при включённой автопрокрутке
* Поведение аналогично обычному режиму браузера
